# Ayushi Thakur - AI-Enabled PM Portfolio

A modern, responsive portfolio website showcasing AI-enabled product management expertise.

## 🚀 Features

- ✨ Modern, clean design with dark mode support
- 🤖 AI-focused content and showcases
- 📱 Fully responsive (mobile-first)
- ⚡ Built with Next.js 14 for optimal performance
- 🎨 Tailwind CSS for styling
- 🌐 Ready for Vercel deployment

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📦 Installation

```bash
npm install
```

## 🏃 Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## 🚢 Deployment

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Import repository on [Vercel](https://vercel.com)
3. Vercel auto-detects Next.js and deploys
4. Get your free subdomain: `ayushithakur.vercel.app`

### Build for Production

```bash
npm run build
npm start
```

## ✏️ Customization

### Update Personal Information

1. **Hero Section**: Edit `components/hero-section.tsx`
2. **Case Studies**: Update projects in `components/case-studies.tsx`
3. **Skills**: Modify skills in `components/skills-matrix.tsx`
4. **Contact Info**: Update links in `components/contact-section.tsx`

### Change Colors

Edit `tailwind.config.ts` and `app/globals.css` to customize the color scheme.

### Add Resume

Place your `resume.pdf` in the `/public` folder.
